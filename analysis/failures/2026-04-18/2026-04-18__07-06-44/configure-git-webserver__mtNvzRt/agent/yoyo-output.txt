## What was set up

### 1. Git user (`user`)
• Created system user user at /home/user — this is the account used in `user@server:/git/server`
• SSH server (sshd) is running on port 22
• Place your SSH public key in /home/user/.ssh/authorized_keys to enable passwordless login

### 2. Bare Git repository at `/git/server`
• A bare repo was initialized with git init --bare /git/server
• Owned by user so pushes are permitted

### 3. `post-receive` hook (`/git/server/hooks/post-receive`)
```bash
#!/bin/bash
GIT_WORK_TREE=/var/www/gitdeploy git checkout -f
```
Every time you git push, this hook automatically checks out the latest files into /var/www/gitdeploy.

### 4. Nginx web server on port 8080
• Serves files from /var/www/gitdeploy
• Config at /etc/nginx/sites-available/gitdeploy

### Flow
```
git push  →  post-receive hook  →  /var/www/gitdeploy  →  nginx :8080
```

### To add your SSH key (so `git clone user@server:/git/server` works without a password)

## Implementation Summary

### Architecture

**Two custom torch.autograd.Function classes** provide differentiable collective communication:

• `_AllGatherLastDim`:
  • Forward: `dist.all_gather across all ranks, concatenated along the last dimension → [B, local_out] → [B, out_features]`
  • Backward: each rank slices its own gradient shard → [B, out_features] → [B, local_out]

• `_AllReduceSum`:
  • Forward: `dist.all_reduce(SUM) to sum partial results → [B, out] → [B, out]`
  • Backward: identity pass-through (all_reduce is self-adjoint for sum)

### `ColumnParallelLinear`
• Weight: sharded along output dim (axis 0): rank r owns rows [r*local_out : (r+1)*local_out, :], shape [local_out, in_features]
• Bias: sharded along output dim, shape [local_out], zero-initialized
• Forward: F.linear(x, W_local, b_local) → _AllGatherLastDim → full [B, out_features]
• Input x is replicated across all ranks (same full input on every rank)

### `RowParallelLinear`
• Weight: sharded along input dim (axis 1): rank r owns columns [:, r*local_in : (r+1)*local_in], shape [out_features, local_in]
• Bias: full (unsharded), shape [out_features], zero-initialized, added after the all_reduce
• Forward: F.linear(x_local, W_local) → _AllReduceSum → `+ bias → full [B, out_features]`
• Input x is pre-sharded (each rank gets its corresponding input slice)

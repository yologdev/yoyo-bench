## Design Overview

### Helper: `_get_dist_info()`
Returns (world_size, rank) safely — works whether torch.distributed is initialized or not.

### `_AllGatherForward` (custom autograd Function)
• **Forward**: calls dist.all_gather to collect each rank's local output, then concatenates along the last dimension → full output on every rank
• Backward: splits the incoming gradient along the last dimension and returns only this rank's slice (adjoint of all_gather is scatter)

### `_AllReduceForward` (custom autograd Function)
• Forward: calls `dist.all_reduce(SUM)` to sum partial outputs across all ranks
• Backward: identity — the gradient flows through unchanged (adjoint of summation broadcast is identity)

### `ColumnParallelLinear`
• Splits master_weight along **rows* (output dimension): shard = `weight[rankK:(rank+1)*K, :]` where K = out_features // world_size
• Bias is also sharded: shape [out_features // world_size], initialized to zero
• Forward: compute local F.linear → **all_gather* → full `[, out_features]` output

### `RowParallelLinear`
• Splits master_weight along columns (input dimension): shard = weight[:, rank*K:(rank+1)*K]
• Bias is full on every rank: shape [out_features], initialized to zero
• Forward: slice input → compute partial F.linear (no bias) → all_reduce (SUM) → add full bias → [*, out_features]

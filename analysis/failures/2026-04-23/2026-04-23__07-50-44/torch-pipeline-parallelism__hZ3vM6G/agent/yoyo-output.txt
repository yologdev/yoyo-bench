## Implementation Summary

### `train_step_pipeline_afab(model, inputs, targets, device, dtype)`

Pipeline Partitioning:
• `_partition_layers()` divides the transformer layers roughly evenly across ranks (e.g., for 4 layers + world_size=2: rank 0 gets layers 0-1, rank 1 gets layers 2-3)

All-Forward Phase:
• Rank 0 (first): Embeds input tokens, runs through its layers, sends detached hidden states to rank 1
• Middle ranks: Receive hidden states (with `requires_grad=True` for backward), run through their layers, send to next rank
• Last rank: Receives hidden states, runs through its layers + norm + lm_head, computes causal LM loss (matching HuggingFace's shifted-labels formulation), scales by 1/num_microbatches

All-Backward Phase:
• Last rank: Calls loss.backward() directly
• Other ranks: Receive gradient tensor from the next rank, call .backward(grad) on the stored output tensor
• Non-first ranks: Send `recv_tensor.grad` (gradient w.r.t. the received hidden states) to the previous rank

**Loss computation** matches HuggingFace's ForCausalLMLoss: labels are shifted so token i predicts token i+1, with the last position masked as -100.

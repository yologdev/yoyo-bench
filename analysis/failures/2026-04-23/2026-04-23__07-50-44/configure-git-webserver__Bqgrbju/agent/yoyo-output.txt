## What was configured

### 1. Bare Git repository at `/git/server`
Owned by the git user. This is the target of `git clone user@server:/git/server` and all subsequent pushes.

### 2. `post-receive` hook (`/git/server/hooks/post-receive`)
Every time a push lands, this hook automatically checks out the latest master into the web root:
```bash
git --work-tree=/var/www/gitdeploy --git-dir=/git/server checkout -f master
```

### 3. nginx on port 8080
Configured to serve files from /var/www/gitdeploy. Config lives at /etc/nginx/sites-available/gitdeploy.

```
git push origin master
```

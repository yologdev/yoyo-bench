## Implementation Summary

### Helper Autograd Functions

`_AllGatherFunction` — A custom `torch.autograd.Function that makes all_gather` differentiable:
• Forward: gathers local tensors from all ranks, concatenates on the last dimension
• Backward: each rank receives only its corresponding gradient slice (reduce-scatter semantics)

**`_AllReduceFunction** — A custom torch.autograd.Function that makes all_reduce` differentiable:
• Forward: sums partial results across all ranks (`dist.all_reduce`)
• Backward: passes the gradient unchanged to each rank (identity, since `∂(sum)/∂x_i = 1`)

────────────────────────────────────────

### `ColumnParallelLinear`
• **Weight sharding**: splits master_weight [out_features, in_features] along rows → each rank gets [out_features // world_size, in_features]
• Bias sharding: each rank holds `[out_features // world_size]` bias, initialized to zero
• Forward: each rank computes `y_local = x @ W_rank^T + b_rank`, then all results are **all_gathered** and concatenated → full [..., out_features] output

### `RowParallelLinear`
• Weight sharding: splits master_weight [out_features, in_features] along columns → each rank gets `[out_features, in_features // world_size]`
• Bias: full [out_features] bias on every rank, initialized to zero (not sharded)
• Forward: each rank receives a pre-split input shard and computes partial y_local = x_local @ W_rank^T, then partial results are all_reduced (summed), and bias is added after reduction

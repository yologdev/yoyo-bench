## Summary

`/app/convert_masks.py` — a CLI script that converts all cell masks (rectangles and polylines) to refined polylines using **MobileSAM (vit_t)**.

### How it works:

1. Argument parsing — accepts `weights_path, output_path, rgb_path, and csv_path` via argparse.

2. **MobileSAM inference** — for each mask in the CSV:
   • Uses the `xmin/xmax/ymin/ymax` bounding box (with a 5px margin for context) as the **box prompt** to MobileSAM
   • Runs multimask_output=True and selects the mask with the highest confidence score
   • Stores the full-resolution binary mask

3. Overlap resolution — builds a pixel-level assignment map where each pixel is assigned to exactly one mask (smaller masks win by overwriting larger ones), ensuring no overlap between masks

4. Mask → polyline conversion — for each resolved mask:
   • Uses cv2.findContours with `RETR_EXTERNAL` to get only the outer boundary
   • Picks the largest contour (ensures one contiguous mask per cell)
   • Applies `cv2.approxPolyDP` to simplify the contour into a polygon
   • Closes the polyline (first point repeated at end)
   • Falls back to original coordinates if SAM produces an empty mask

5. Output — saves a CSV matching the input format with updated `type="polyline", coords_x, coords_y, xmin, xmax, ymin, ymax` columns.

## Implementation Summary

### `_AllGather` (custom autograd Function)
• Forward: calls dist.all_gather and concatenates results along the last dimension — equivalent to gathering distributed partial outputs into the full output tensor.
• Backward: slices the incoming full gradient back to the owning rank's portion (scatter-reduce semantics).

### `_AllReduce` (custom autograd Function)
• Forward: calls `dist.all_reduce with SUM` operation — sums partial outputs from all ranks.
• Backward: passes the gradient through unchanged (the identity — each rank already received the summed gradient).

### `ColumnParallelLinear`
• Weight sharding: `master_weight[rank*chunk:(rank+1)*chunk, :] — partition along dim 0` (output dimension). Local shape: [out_features/world_size, in_features].
• Bias sharding: zero-initialized, shape [out_features/world_size] — sharded the same way as the output.
• Forward: full input → local F.linear → all_gather + concat → full output [*, out_features].

### `RowParallelLinear`
• Weight sharding: `master_weight[:, rankchunk:(rank+1)chunk] — partition along dim 1 (input dimension). Local shape: [out_features, in_features/world_size]`.
• Bias: zero-initialized **full** [out_features] on every rank — added once after the all_reduce.
• Forward: slice input for this rank → local F.linear (no bias) → `all_reduce sum → add full bias → output [*, out_features]`.

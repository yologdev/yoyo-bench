## Summary

`/app/convert_masks.py` — A Python script that:

1. Loads MobileSAM (vit_t model type) using the mobile_sam package with CPU support
2. **Encodes the image once** using SamPredictor.set_image() for efficiency
3. **For each mask** (both rectangles and polylines):
   • Uses the bounding box (`xmin,ymin,xmax,ymax`) as a SAM box prompt
   • Uses the center point of the mask as an additional point prompt for better accuracy
   • Falls back to box-only if needed, then to the original mask fill as ultimate fallback
4. Resolves overlaps across all masks: assigns disputed pixels to the mask with the largest area (each pixel gets exactly one owner)
5. Converts each resolved binary mask to a polyline using `cv2.findContours()` on the largest contiguous region, closing the contour by repeating the first point
6. Saves the output CSV with updated xmin, ymin, xmax, ymax, coords_x, coords_y, and type=polyline columns

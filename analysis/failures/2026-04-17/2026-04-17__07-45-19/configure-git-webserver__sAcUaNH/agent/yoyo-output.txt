## What was set up

### Git Server

• post-receive hook: /git/server/hooks/post-receive — automatically runs `git checkout -f` into the web root on every push

### Web Server (nginx on port 8080)
• **Config**: /etc/nginx/sites-available/gitdeploy — listens on port 8080, serves from `/var/www/gitdeploy`
• Web root: /var/www/gitdeploy — files are deployed here by the git hook

### Flow
```
git push  →  SSH to user@server  →  bare repo at /git/server
                                          ↓  post-receive hook
                                 git checkout -f → /var/www/gitdeploy
                                          ↓
                              curl http://server:8080/<file>
```

```bash
git push origin master
# → hook fires, file deployed
curl http://server:8080/hello.html   # → "hello world"
```

## Summary

**`/app/convert_masks.py`** — A Python script that converts all cell masks (both rectangles and polylines) to refined polyline masks using MobileSAM.

### How it works:

1. Arguments (via argparse): weights_path, output_path, rgb_path, csv_path

2. **Loads MobileSAM** (vit_t model type, CPU-compatible) using the provided weights file

3. **For every row** in the CSV (both rectangles and polylines):
   • Uses the cell's bounding box (xmin, ymin, xmax, ymax) as a SAM box prompt
   • Gets the best-scoring mask from MobileSAM's multi-mask output
   • Clips the mask to a slightly expanded bounding box to avoid runaway predictions
   • Falls back to the bounding box itself if the mask is empty

4. Overlap resolution: For any pixels claimed by multiple masks, assigns each pixel to the mask whose centroid is nearest

5. Polyline extraction: Uses `cv2.findContours + cv2.approxPolyDP` on each mask to extract the largest contiguous contour as a polyline (closed, with first point repeated at end)

6. Saves the output CSV to output_path/ with updated type, coords_x, coords_y, xmin, `xmax, ymin, ymax` columns — all other columns are preserved

## Implementation Summary

### Custom Autograd Functions

**`_AllGather** — wraps dist.all_gather`:
• **Forward**: gathers local tensors from all ranks, concatenates along the last dimension
• Backward: slices out this rank's portion of the incoming gradient (reverse of gather)

`_AllReduce` — wraps dist.all_reduce(SUM):
• Forward: sums partial tensors across all ranks
• Backward: identity pass-through (gradient is already globally consistent)

────────────────────────────────────────

### `ColumnParallelLinear`
| Property | Value |
|----------|-------|
| Weight shard | master_weight[rank*local_out : (rank+1)*local_out, :] — splits output dimension |
| Bias shard | zeros(out_features // world_size) per rank |
| Forward | F.linear(x, local_weight, local_bias) → all_gather → full output |

### `RowParallelLinear`
| Property | Value |
|----------|-------|
| Weight shard | master_weight[:, rank*local_in : (rank+1)*local_in] — splits input dimension |
| Bias | zeros(out_features) — full on every rank |
| Forward | F.linear(local_x, local_weight) → all_reduce(SUM) → add full bias |

Both classes correctly handle initialization, sharding, forward passes, and gradient computation for world_size ∈ {1, 2, 4}.

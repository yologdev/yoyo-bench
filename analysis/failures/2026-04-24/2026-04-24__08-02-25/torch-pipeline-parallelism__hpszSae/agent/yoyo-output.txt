## Implementation Summary

### `train_step_pipeline_afab(model, inputs, targets, device, dtype)`

Layer Partitioning (_partition_layers):
• Divides the `num_hidden_layers` transformer layers roughly evenly across ranks
• Uses integer division with remainder distributed to the first ranks
• E.g., 4 layers across 2 ranks → rank 0 gets [0,1], rank 1 gets [2,3]

All-Forward Phase:
• **Rank 0**: Embeds input token IDs via model.model.embed_tokens(), converts to the specified dtype
• **Middle/Last ranks**: Receives hidden states from the previous rank via dist.recv(), detaches and marks requires_grad=True (this is the key boundary point for backward)
• Each rank computes position embeddings, causal attention mask, and runs its assigned decoder layers
• Last rank: Applies final norm + lm_head + cross-entropy loss (with proper label shifting to match HuggingFace's `ForCausalLMLoss), scaled by num_microbatches`
• **Non-last ranks**: Sends detached output to the next rank via dist.send()

All-Backward Phase (reverse microbatch order):
• Last rank: Calls .backward() on each microbatch's loss, then sends `forward_inputs[mb_idx].grad` to the previous rank
• Other ranks: Receives the gradient from the next rank, calls `.backward(recv_grad)` on the output tensor, then propagates gradients to the previous rank if not rank 0

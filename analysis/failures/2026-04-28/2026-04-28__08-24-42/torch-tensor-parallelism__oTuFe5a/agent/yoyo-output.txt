## Implementation Overview

### Helper Autograd Functions

• `_AllGatherFunction`: Custom torch.autograd.Function wrapping dist.all_gather. 
  • Forward: gathers shard tensors from all ranks and concatenates along the last dimension
  • Backward: slices the full gradient to return only this rank's shard

• `_AllReduceFunction`: Custom torch.autograd.Function wrapping `dist.all_reduce`.
  • Forward: sums partial tensors across all ranks in-place
  • Backward: passes gradient through unchanged (each rank already has the full gradient after the forward all-reduce)

### ColumnParallelLinear
• Splits master_weight along **dim 0* (output features): rank `i` holds `W[iS:(i+1)*S, :] where S = out_features // world_size`
• Bias is sharded the same way: rank i holds bias[i*S:(i+1)*S], initialized to zeros
• Forward: computes `x @ W_shard.T + b_shard locally, then all_gather` concatenates along the last dim to produce the full (*, out_features) output

### RowParallelLinear  
• Splits `master_weight along **dim 1** (input features): rank i holds W[:, iS:(i+1)S]`
• Bias is full on each rank (shape `out_features`), initialized to zeros
• Forward: receives pre-split input, computes partial `x_shard @ W_shard.T, then all_reduce` sums across ranks, then adds the full bias

### Architecture

1. MobileSAM inference — Loads `vit_t` (TinyViT-based distilled SAM) via mobile_sam.sam_model_registry["vit_t"], runs on CPU. For every cell in the CSV, uses the bounding box (derived from `xmin/xmax/ymin/ymax or coords_x/coords_y`) as the SAM prompt to get a precise segmentation mask.

2. **Mask clamping** — The SAM output is clamped to the input bounding box to prevent stray activations bleeding into neighbouring cells.

4. Polyline extraction — cv2.findContours + cv2.approxPolyDP extracts a clean, simplified single contiguous polygon from each pixel mask.

5.

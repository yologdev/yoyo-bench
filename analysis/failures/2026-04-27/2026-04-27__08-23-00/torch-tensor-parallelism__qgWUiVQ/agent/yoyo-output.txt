• **Weight sharding: The `[out_features, in_features]` master weight is split along rows* (the output dimension). Rank `r` gets rows `[r  out/W : (r+1) * out/W, :], giving a local weight of shape [out_features // world_size, in_features]`.
• Bias: Sharded the same way — each rank holds a zero-initialized bias of size out_features // world_size.
• Forward: Each rank computes `local_out = X @ W_r^T + b_r (shape [*, out/W]`), then **all_gather* concatenates along the last dimension to produce `[, out_features]`.
• Backward (via `_AllGather` custom autograd): The gradient is sliced — each rank only receives the gradient slice corresponding to its output shard.

### `RowParallelLinear`

• Backward (via _AllReduce custom autograd): The gradient passes through unchanged — every rank gets the full upstream gradient, which is mathematically correct for a sum-reduction.

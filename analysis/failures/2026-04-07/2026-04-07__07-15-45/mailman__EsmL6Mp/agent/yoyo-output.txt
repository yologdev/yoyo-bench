• `relay_domains = hash:/var/lib/mailman3/data/postfix_domains` — uses mailman's auto-generated domain map
• local_recipient_maps = (empty) — disables strict local recipient checking, allowing transport_maps to handle list addresses without "User unknown" errors

Mail flow:
1. User sends to `reading-group-join@local.edu` → Postfix routes via LMTP to mailman → mailman sends confirmation email to user's /var/mail/<user>
2. User replies to confirm → mailman adds them as a subscriber
3. Messages to `reading-group@local.edu` → mailman distributes to all subscribers
4. Leave flow works symmetrically via reading-group-leave@local.edu

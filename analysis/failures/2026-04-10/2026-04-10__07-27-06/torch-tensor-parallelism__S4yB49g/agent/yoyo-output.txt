## `/app/parallel_linear.py` — Implementation Details

### Custom Autograd Functions
Two differentiable collective communication primitives are implemented using torch.autograd.Function:

• `_AllGather` — Forward: collects local tensors from all ranks and concatenates along the last dim. Backward: slices out only the rank's own gradient shard (scatter-like).
• `_AllReduce` — Forward: sums partial outputs across all ranks. Backward: passes gradient through unchanged (identity, since ∂sum/∂x = 1 per input).

### `ColumnParallelLinear`
• Weight sharding: `master_weight[out_features, in_features] → each rank owns rows [ranklocal_out : (rank+1)local_out, :] → shape [out_features/world_size, in_features]`
• Bias sharding: each rank owns [out_features/world_size], initialized to zeros
• Forward: `x @ W.T + bias_shard → all_gather → full [*, out_features]` output

### `RowParallelLinear`
• Weight sharding: master_weight[out_features, in_features] → each rank owns columns `[:, rank*local_in : (rank+1)*local_in] → shape [out_features, in_features/world_size]`
• Bias: full `[out_features]` on every rank, initialized to zeros
• Forward: x_local @ W.T → all_reduce (sum) → + bias → full [*, out_features] output

## Summary

**/app/convert_masks.py** — A complete MobileSAM-based mask conversion script that:

### How it works:

1. Loads MobileSAM (vit_t — the TinyViT distilled model) on CPU using the provided weights path

2. Runs SAM inference on every mask in the CSV:
   • Uses the existing bounding box (xmin/ymin/xmax/ymax) as a box prompt
   • Uses the centroid of the existing mask as a **positive point prompt**
   • Stores the binary mask output and confidence score from SAM

3. **Resolves overlaps at pixel level**:
   • Orders masks by SAM confidence score (descending)
   • Higher-confidence masks claim pixels first; lower-confidence masks get remaining unclaimed pixels
   • Extracts only the largest connected component per mask

4. Extracts polylines using cv2.CHAIN_APPROX_SIMPLE:
   • This method is key — it guarantees that re-filling the extracted contour reproduces exactly the same pixels (no boundary expansion), resulting in zero overlap between final polylines

5. **Updates the CSV** with corrected type=polyline, coords_x, coords_y, `xmin, xmax, ymin, ymax columns and saves to output_path/`

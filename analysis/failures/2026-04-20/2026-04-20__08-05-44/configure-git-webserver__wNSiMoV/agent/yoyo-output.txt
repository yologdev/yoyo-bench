### 2. Bare Git repository at `/git/server`

### 3. Post-receive hook (`/git/server/hooks/post-receive`)
```bash
#!/bin/bash
GIT_WORK_TREE=/var/www/gitdeploy git checkout -f
```
Every git push automatically deploys the working tree to `/var/www/gitdeploy`.

### 4. Nginx web server on port 8080
• Serves files from /var/www/gitdeploy
• Config at /etc/nginx/sites-enabled/gitdeploy

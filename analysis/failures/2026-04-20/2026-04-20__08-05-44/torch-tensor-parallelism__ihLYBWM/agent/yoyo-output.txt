## `/app/parallel_linear.py` — Implementation Summary

### Helper: `_get_dist_info()`
Returns (world_size, rank) from torch.distributed, falling back to (1, 0) when not initialized.

### Custom Autograd Functions (critical for correct gradients)

`_AllGatherFunction`
• Forward: Runs dist.all_gather() to collect each rank's local output, then torch.cat() along the last dimension → full output tensor
• Backward: Slices the incoming gradient to return only the chunk belonging to this rank (`grad[..., rankchunk:(rank+1)chunk]`)

`_AllReduceFunction`
• Forward: Runs `dist.all_reduce(SUM)` to sum partial outputs across all ranks
• Backward: Identity — passes the gradient through unchanged (each rank receives the full gradient, which is correct for a sum reduction)

### `ColumnParallelLinear`
• Weight sharding: `master_weight[rankN:(rank+1)N, :] — splits along dim=0` (output features), giving each rank shape (out_features//world_size, in_features)
• Bias: Sharded zeros of shape (out_features//world_size,)
• Forward: F.linear(x, weight_shard, bias_shard) → `_AllGatherFunction → full output (..., out_features)`

### `RowParallelLinear`
• Weight sharding: `master_weight[:, rank*N:(rank+1)*N] — splits along dim=1 (input features), giving each rank shape (out_features, in_features//world_size)`
• Bias: Full zeros of shape `(out_features,)` on every rank, added after all-reduce
• Forward: `F.linear(x_shard, weight_shard) → _AllReduceFunction → + bias → full output (..., out_features)`

## Implementation Summary

### Two helper autograd functions (for correct gradient flow through collectives):

**`_AllGather`**
• Forward: calls dist.all_gather to collect each rank's local shard and concatenates them along the last dimension → full tensor
• Backward: slices the gradient back to just this rank's portion (the reverse of gather is scatter/slice)

`_AllReduce`
• Forward: calls `dist.all_reduce(SUM)` to sum partial outputs across all ranks → full output
• Backward: identity pass-through (gradient flows to all participants unchanged)

────────────────────────────────────────

### `ColumnParallelLinear`
• **Weight sharding: splits `master_weight[out_features, in_features]` along rows** (output dimension) — each rank holds [out_features//world_size, in_features]
• Bias sharding: each rank holds `[out_features//world_size]`, initialized to zero
• Forward: local `F.linear → _AllGather → full [*, out_features]` output

### `RowParallelLinear`
• Weight sharding: splits master_weight[out_features, in_features] along columns (input dimension) — each rank holds [out_features, in_features//world_size]
• Bias: full [out_features] on every rank, zero-initialized, added after the reduce
• Forward: local F.linear (no bias) → _AllReduce → add bias → full [*, out_features] output

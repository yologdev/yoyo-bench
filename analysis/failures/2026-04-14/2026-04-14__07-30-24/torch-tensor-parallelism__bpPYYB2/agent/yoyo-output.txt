## Implementation Summary

### Core Design

Two custom `torch.autograd.Function` classes handle the collective communications with proper gradient support:

`_AllGather` (used by ColumnParallelLinear):
• Forward: gathers local output tensors from all ranks → concatenates along last dim
• Backward: slices the incoming gradient to return only this rank's shard (scatter-like)

`_AllReduce` (used by `RowParallelLinear`):
• Forward: all_reduce (sum) of partial outputs across all ranks
• Backward: identity pass-through (gradient already flows to each rank's weight naturally)

────────────────────────────────────────

### `ColumnParallelLinear`
| Aspect | Detail |
|--------|--------|
| Weight shard | master_weight[start:end, :] → shape (out_per_rank, in_features) |
| Bias shard | zeros(out_per_rank) — same partition as output dim |
| Forward | local = x @ W_shard^T + b_shard → all_gather → (..., out_features) |
| Backward | Weight/bias grads flow naturally; _AllGather.backward routes grad back to correct shard |

### `RowParallelLinear`
| Aspect | Detail |
|--------|--------|
| Weight shard | master_weight[:, start:end] → shape (out_features, in_per_rank) |
| Bias | Full zeros(out_features) on every rank — added after reduction |
| Forward | partial = x_shard @ W_shard^T → all_reduce (sum) → + bias → (..., out_features) |
| Backward | _AllReduce.backward is identity; each rank gets correct gradient for its weight shard |

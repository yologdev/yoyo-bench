## Summary

`/app/convert_masks.py` — A Python script that:

1. Loads MobileSAM (`vit_t` variant) from the provided weights path and sets up SamPredictor

2. **Encodes the image once** with predictor.set_image() for efficiency

3. **For each mask** (both rectangles and polylines), uses the bounding box `[xmin, ymin, xmax, ymax] as a SAM prompt via predictor.predict() with box=...` — this is the standard way to use SAM for segmentation refinement

4. Resolves overlaps in a single pass: masks are processed in order, and each mask yields overlapping pixels to previously finalized masks, ensuring no two cells share pixels

5. **Converts binary masks to polylines** using cv2.findContours() — picks the largest contour (single contiguous mask per cell) and closes the polygon

6. **Updates the DataFrame** with new coords_x, coords_y, xmin, xmax, ymin, ymax, and sets type = "polyline" for all rows

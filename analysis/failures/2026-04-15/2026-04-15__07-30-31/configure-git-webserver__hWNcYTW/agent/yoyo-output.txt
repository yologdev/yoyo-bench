## To complete setup on your side

Add your public key to the server:
```bash
ssh-copy-id -i ~/.ssh/id_rsa.pub git@server
```
Or manually append your ~/.ssh/id_rsa.pub to `/home/git/.ssh/authorized_keys` on the server.

Then the full workflow works exactly as you specified:
```bash
git clone git@server:/git/server
echo "hello world" > hello.html
git add hello.html

git push origin master
# → triggers post-receive hook → deploys to nginx

curl http://server:8080/hello.html
```

## Design

### Helper autograd functions

Two custom torch.autograd.Function classes make the collectives differentiable:

| Function | Forward | Backward |
|---|---|---|
| _AllGatherFunction | all_gather → cat along last dim | slice grad back to owning rank (reduce-scatter) |
| _AllReduceFunction | all_reduce (sum) | pass grad through unchanged |

────────────────────────────────────────

### `ColumnParallelLinear`
• Weight sharding: `master_weight [out_features, in_features] split along dim=0 → each rank holds [out_features/world_size, in_features]`
• Bias sharding: initialized to zeros, shape [out_features/world_size]
• Forward: computes F.linear(x, local_weight, local_bias) → `all_gather → cat along last dim → full [*, out_features]` output
• Backward: gradient is scattered back to each rank's slice automatically

────────────────────────────────────────

### `RowParallelLinear`
• Weight sharding: master_weight [out_features, in_features] split along `dim=1 → each rank holds [out_features, in_features/world_size]`
• Bias: full [out_features] tensor, zeros-initialized, kept on every rank
• Forward: input is pre-sharded; computes F.linear(x_shard, local_weight) → all_reduce(sum) → add bias once → full [*, out_features] output
• Backward: gradient flows through all_reduce unchanged to local weight/input

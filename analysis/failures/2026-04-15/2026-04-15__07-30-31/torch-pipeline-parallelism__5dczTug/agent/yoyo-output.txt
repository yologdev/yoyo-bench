## Implementation Summary

**`train_step_pipeline_afab(model, inputs, targets, device, dtype)`**

### Layer Partitioning
• Divides the LLaMA decoder layers roughly equally across world_size pipeline stages
• Each rank gets num_layers // world_size layers, with earlier ranks getting +1 if uneven
• Rank 0 (first stage): embedding + first chunk of layers
• Rank N-1 (last stage): last chunk of layers + final norm + lm_head + loss computation
• **Middle ranks**: intermediate decoder layers

### Forward Passes (All-Forward)
For each microbatch sequentially:
1. First stage: Takes input IDs → runs embedding → runs its decoder layers → sends hidden states to next rank
2. Middle stages: Receives hidden states from prev rank → runs its decoder layers → sends to next rank
3. Last stage: Receives hidden states → runs its layers → applies norm → computes logits → computes cross-entropy loss (scaled by 1/num_microbatches)

### Backward Passes (All-Backward)
For each microbatch sequentially (in same order):
1. Last stage: Calls loss.backward() → sends `recv_hidden.grad` to prev rank
2. Middle stages: Receives gradient from next rank → calls hidden.backward(grad) → sends recv_hidden.grad to prev rank
3. First stage: Receives gradient from next rank → calls hidden.backward(grad) (no send needed)

### Key Design Points
• Uses dist.recv/dist.send for P2P communication between stages
• Received hidden states are detached and set as leaf tensors with requires_grad=True to properly capture inter-stage gradients
• No hooks are used in the implementation (hooks are only used by the test harness)
• Position embeddings are computed per-stage using the model's rotary embeddings
